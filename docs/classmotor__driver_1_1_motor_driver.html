Detailed Description

This class implements a motor driver for a generic motor attached to the nucleo and its H bridge shield.

This class takes pins put in by the user to set up a motor timer and motor channel. The pins used by this version are PA10, PB4 and PB5 with timer channel 3 for motor 1 and PC1, PA0 and PA1 with timer channel 5 for motor 2 .
