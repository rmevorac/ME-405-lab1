Detailed Description

This class implements a motor driver for an ME405 kit.

This class handles the set up and data managment of the encoders through utilzing the B7/B6 and C7/C6 pins and their respective and corresponding timers, timers 4 and 8. This class also handles the encoder logic math to handle overflow, underflow and detection of directional changes from the motor.
